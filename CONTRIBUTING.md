# Contributing to Not A Company

Thank you for your interest in contributing to this living manifesto!

## How to Contribute

This manifesto evolves through community input. Here's how you can contribute:

### 1. **Suggest Edits to the Core Manifesto**
If you want to propose changes to the main README.md:
- Fork this repository
- Make your changes in a new branch
- Open a Pull Request with:
  - Clear description of what you're changing and why
  - How it aligns with or extends the core principles

**Review Process:** Changes to the core manifesto will be carefully reviewed to maintain coherence and vision.

### 2. **Share Ideas for Consideration**
Have thoughts that might not fit directly into the manifesto yet?
- Open an Issue with the `idea` label
- Describe your thinking and how it relates to the manifesto
- Engage in discussion with the community

### 3. **Report Issues or Unclear Sections**
Found something confusing or want clarification?
- Open an Issue with the `question` or `clarification` label
- Point to the specific section
- Explain what's unclear

## Version History

All changes are tracked through Git's native version control:
- View the commit history to see how the manifesto has evolved
- Click the "History" button on any file in GitHub to see its evolution
- Use `git log` or `git blame` to trace specific changes

## Guidelines

When contributing, please:
- **Stay True to Core Principles**: Autonomy, transparency, ownership, and extreme ownership
- **Be Specific**: Vague suggestions are hard to act on
- **Explain Your Reasoning**: Help us understand your perspective
- **Respect the Vision**: This is a living document, but it has a clear direction

## Questions?

Not sure if your idea fits? Open an issue and let's discuss!

---

**Remember:** This isn't just a document—it's a movement. Your contribution helps shape what's possible.
